## Lessons

- Jinja2 parses `${#var}` as a comment opener `{#` — always wrap bash scripts in `{% raw %}` blocks in Ansible
- Ansible handlers are fragile: if ANY task fails before the play ends, ALL queued handlers are dropped. Use `meta: flush_handlers` after critical config tasks
- Go 1.22 ServeMux `"/"` catch-all can shadow parameterized routes like `"GET /api/.../tokens/{id}/timeline"` — guard the catch-all handler against `/api/` paths
- The alive-server binary runs from the repo dir, not `/usr/local/bin/` — always check `systemctl cat` for the actual ExecStart path
- `curl -sf` hides HTTP error codes (401 becomes exit 22 → `|| HTTP_CODE="000"`) — drop `-f` for registration flows where the real status matters
- `pd-standard` HDD writes at ~12.7 MB/s — copying a 6GB image takes 8 minutes. Keep base images uncompressed on disk to avoid the decompress step
