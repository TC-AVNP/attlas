## Lessons

- When building a dramatic visual redesign, always present it behind a toggle first. Aesthetic preferences are deeply personal — what looks amazing in a prototype may not survive contact with the person who uses the tool every day.
- Tailwind utility classes can conflict with custom CSS class names. `.invert` in Tailwind applies a CSS filter while a custom `.invert` class for reverse-video text does something completely different. Namespace custom classes (e.g. `.crt-reverse`) to avoid collisions.
- Lazy-loading with `React.lazy()` is the right way to gate a beta UI — the CRT assets (CSS, audio, components) only download when the user opts in, keeping the default bundle lean.
- AFM stores uploaded files at `/var/lib/afm/files/` — useful for transferring design assets from browser to the build environment.
