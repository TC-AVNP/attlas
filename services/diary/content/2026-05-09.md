## Lessons

- AltStore PAL (EU DMA) requires $99/year — the free path is AltStore Classic only, with 7-day auto-refresh
- The WKWebView + JS bridge pattern (Capacitor/Ionic) is Apple-approved for web content with native API access
- `replace_all: true` on SQL edits is dangerous when queries have different column prefixes — always make targeted edits
- Human descriptions should be functional paragraphs split by concern, not marketing copy
- Never use Claude's auto-memory system — store in knowledge graph or skill files (Commandment #2)
- Effort logging per-project-lump is useless — must be per-feature with JSONL timestamp analysis
- When backfilling lifecycle timestamps, set them via SQLite directly since MCP auto-sets only work on future transitions
- When a UI component is used on multiple pages, extract it into a shared component immediately — duplicating the card between Kanban and Completed caused a bug where the Completed page was missing the APP ribbon
- Iterative UI design with live feedback produces better results than a single spec — the user refined the card layout 8 times in 30 minutes, each iteration improving on specific pain points
