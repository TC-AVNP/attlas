## Lessons

- AltStore PAL (EU DMA) requires $99/year paid Apple Developer account — the free account path is AltStore Classic only
- Apple notarization is an automated security scan, not human review — but still requires paid account to access App Store Connect
- The WKWebView + JS bridge pattern (Capacitor/Ionic) is Apple-approved for loading web content with native API access — thousands of App Store apps use it
- `replace_all: true` on SQL edits is dangerous when two queries have different column prefixes (p.col vs col) — always make targeted edits
- Human descriptions should be functional paragraphs split by concern, not marketing copy
- Never use Claude's auto-memory system — store everything in the knowledge graph or skill files
