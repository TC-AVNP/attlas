React frontend replaced entirely: sidebar layout, markdown rendering with GFM, embedded 3D/2D schematics on specific pages, journal section with blog-style entries. **Lines:** 1-945

## Lessons

- Preserving original GLB model materials (just adding emissive tint) looks far better than replacing all materials with flat color
- Html labels in Three.js don't scale with camera distance by default — show them only on hover to avoid cluttering the viewport
- react-markdown with remark-gfm handles tables well but needs custom components for proper dark-theme styling
- Internal wiki links need React Router navigation (not full page reload) — handle in the markdown link component
