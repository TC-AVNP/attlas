## Project summaries

### davids-checklist

Built a complete task handover web app from scratch. Go backend with SQLite, React frontend behind Google OAuth. Admin creates handovers with task lists, shares with a friend who ticks them off. Telegram notifications sent via OpenClaw when tasks are completed. Archive feature to hide done handovers. Deployed to david.attlas.uk with Caddy routing. Significant debugging around DNS, OAuth callbacks, and authorization.
**Lines:** 29-857 (session 1)

### petboard

Added a markdown export button to the project overview. Exports project details and selected features as formatted markdown for sharing outside the app.
**Lines:** 2-105 (session 2)

## Lessons

- New services need the full deployment checklist: Caddy snippet, DNS record, alive-server registration, install/uninstall scripts — skipping steps causes "domain not reachable" issues
- OAuth callback URLs must be pre-registered in the Google Cloud Console before testing
