## Lessons

- Always check Secret Manager IAM bindings when adding new secrets to Terraform.
- readlink -f on npm-installed binaries resolves to the .mjs entry point, not the package root.
- Cloudflare DNS propagates in seconds — no waiting needed.
- Old session cookies survive server rewrites if the session secret persists on disk.
- Caddy runs as its own user — can't read files in user home dirs, proxy through app server instead.
- Session transcripts contain secrets — must redact before committing to git.
- Checking binary existence isn't enough for install state — use an explicit state file.
