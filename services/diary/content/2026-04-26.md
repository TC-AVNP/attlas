### Terminal session rename (~15min)

Added the ability to rename tmux sessions from the terminal detail page
(`attlas.uk/services/details/terminal`). Previously only create and kill
were supported.

**Backend** (`internal/services/services.go`):
- New `HandleTerminalRename` handler at `POST /api/services/terminal/rename`
- Accepts `{"name": "old", "new_name": "new"}`, validates both with the
  existing `^[a-zA-Z0-9_-]{1,32}$` regex
- Runs `tmux -L attlas rename-session -t =<old> <new>` via the shared
  `tmuxCmd()` helper

**Frontend** (`Terminal.jsx`):
- Added a visible "rename" button in each session row's action area
- Clicking it turns the session name into an inline editable input
- Enter/blur confirms, Escape cancels
- Toast feedback on success/failure, list refreshes after rename

**Route** (`main.go`):
- Registered `POST /api/services/terminal/rename → services.HandleTerminalRename`

Follows the same pattern as `HandleTerminalKill` — validate, shell out
to tmux, return success/error JSON.

### Mobile keyboard UX overhaul (~30min)

The mobile keyboard overlay (`ttyd-mobile-keyboard.html`) was unusable
on phones — keys were enormous, the keyboard showed by default eating
half the screen, and speech-to-text caused triple input because the
native IME fought with xterm's hidden textarea.

**Changes:**

- **Keyboard hidden by default** — removed the CSS media query that
  forced `display:flex` on mobile. Both keyboards start hidden.
- **Two toggle buttons** — replaced the single 132px toggle with two
  52px side-by-side circles: "Aa" (text input modal) and "⌨" (custom
  HTML keyboard).
- **Text input modal** — the "Aa" button opens a full-screen textarea
  where speech-to-text and native typing work cleanly. Press Send to
  push all text to the terminal in one shot, avoiding duplicate input.
- **Viewport zoom lock** — injected
  `<meta name="viewport" ... maximum-scale=1,user-scalable=no>` into
  `<head>` via `install.sh` so the browser never zooms on input focus.
- **Scaled-down sizing** — keys from 51px→16px font, bar from 36px→13px,
  toggles from 132px→52px, gaps from 4px→2px, keyboard height 40vh→34vh.
