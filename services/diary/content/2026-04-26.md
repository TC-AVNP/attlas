## Lessons

- Viewport meta must be in HTML head, not set from JS. Browsers read viewport constraints before running scripts, so setting maximum-scale=1 from JS is a race condition.
- Never let the native keyboard interact directly with xterm's hidden textarea. IME composition events fire multiple times causing duplicate input. Isolate input into a separate textarea and send the final text via the JS API.
- Know what process owns stdin. Sending "exit" to a terminal running Claude Code CLI doesn't exit the shell — need /exit for Claude Code first, then exit for the shell.
- Manual confirmation beats heuristic detection for async processes. Polling cursor position for silence was fragile; a simple "click when ready" is more reliable and less code.
- tmux has no native session metadata. A flat JSON file keyed by session name is the simplest external storage for descriptions — no database, survives restarts, easy to inspect.
- git-filter-repo is the right tool for purging large files from history. Always backup first and verify commit count plus working tree before force-pushing.
